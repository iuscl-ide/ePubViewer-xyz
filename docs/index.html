ePub Viewer is an application for quickly preview and then read book files in ePub format. It can also quickly open MD (Markdown) files with the same quality as in GitHub.

This is the common epubviewer.xyz and mdviewer.com landing page

The need for this application came from the fact that other applications are either too slow, do not have a good render quality or are extracting the book on disk and do not delete it after (such as Edge). Also, for other applications it is not easy to change the font and the size for the whole book, there are always some parts which remain different from the others. So, the idea was to have an application which can be used to quickly open, preview and close a book by double click on it in a folder with a lot of books. However, if a book will be chosen to store its customizations, the bookmarks and the display settings will be saved on disk in the operating system provided location.

Features
  • Quickly open and read ePub books
  • Nothing is written on disk by default, the ePub file is opened and processed entirely in memory
  • Customizable font and style can be forced for all elements (not only paragraphs and headers)
  • For security, no external links are followed by the application, the embedded ones are ignored, and the ones in page are not opened even if the user clicks on them
  • If the user selects the option to save information on disk, then and only then the customizations done for a book (including bookmarks) will be stored
  • The application can also open MD (Markdown) files, and render them either identical to GitHub, or in a more "Google" style. First right click on a .md file and open it with ePubViewer and then the .md files can be associated with the application with an option from the menu
Notes
  • For ePubs no Internet links are opened, but for MD the links are opened, the ones inside the MD will be downloaded and the ones clicked by the user will be opened in an external browser
  • No MathML is rendered if the book contains mathematical expressions; it was implemented, it worked OK but the size of the library was too big to be included for this functionality
  • For very large books (> 50MB) it will take a few seconds to open; the application does not display a progress indicator because usually the books are not so big
  • No versions were done/tested for 32 bits platforms, only 64 bits platforms are available
Download

ePubViewer is written entirely in Electron, and so it can work on different platforms, download the version corresponding to the user platform. Currently, as the application is still in testing, only the Windows one is available:

  • Windows 64 bits: download new version 1.0.2
  • Linux 64 bits: not yet available
  • Mac 64 bits: not yet available
